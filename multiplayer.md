# Now you can play multiplayer.

1. The Host creates a Google Sheet and a Google Doc.

2. The Host pastes the Doc ID into Settings!B3.

3. The Host deploys the Code.gs and puts the resulting URL into Settings!B1.

4. Players simply open the index.html?id=MyName&sheet=URL link.

5. When Player A installs an asset: It writes to the Sheet AND appends "[Time] PLAYER A: SYSTEM UPDATE: Installed Courier" to the Doc.

6. When Player B rolls Edge: It appends "[Time] PLAYER B: ROLL: Edge | Result: Weak Hit" to the Doc.

> [!Tip]
> Everyone's game data is just in the Host's Google Sheet and the group's collective game log is in the Host's Google Doc.

7. Profit!
